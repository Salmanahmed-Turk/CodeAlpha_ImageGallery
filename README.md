# CodeAlpha_ImageGallery
This project is a modern and responsive Image Gallery developed using HTML, CSS, and JavaScript. It features a dynamic masonry-style layout that adapts to different screen sizes and provides an interactive browsing experience. Users can filter images by category, search for images, switch between light and dark modes, and view images in a fullscreen lightbox with next/previous navigation and keyboard controls.

The project focuses on creating a clean UI and smooth user interaction using modern frontend design practices and responsive layouts.

## Features
- Responsive masonry grid layout
- Image filtering by categories
- Search functionality
- Lightbox image preview
- Next / Previous navigation
- Keyboard controls
- Dark mode toggle
- Smooth hover animations

## Technologies Used
- HTML5
- CSS3
- JavaScript
